Basic Machine Architecture

Stored Program Computer

  • 컴퓨터 내부에 저장된 일련의 명령

    • 미리 정의된 기본 명령어 세트로 구성

      • 간단한 테스트

      • 데이터 이동

  • 특별한 프로그램(인터프리터)는 각 명령(instruction)을 순서대로 실행

    • 테스트를 사용하여 시퀀스를 통해 제어 흐름을 변경

    • 완료되면 중지

  • Alan turing은 6개의 기본 요소를 사용하여 무엇이든 계산할 수 있음을 증명

Aspects of Languages(언어적 관점에서)

  • Primitive constructs

    • 영어: 단어로 이루어짐

    • 프로그래밍 언어: 숫자, 문자열, 단순 연산자로 이루어짐

Aspects of Languages - Syntax

문장의 형태로는 유효함

  • English

    • “cat dog boy” → 구문상 유효하지 않음

    • “cat hugs boy” → 구문상 유효함

  • Programming Language

    • “Hi” 5 → 구문상 유효하지 않음

    • 3.2 * 5 → 구문상 유효함

Aspects of Languages - Static Semantics

구문적으로 유효한 문자열이 의미를 가지는 것

  • English

    • “I are hungry” → 구문적으로 유효하지만 정적 의미로는 유효하지 않음

  • Programming Language

    • 3.2 * 5 → 구문적으로 유효

    • 3 + “hi” → static semantics 오류

Aspects of Languages - Semantics

정적 의미(static semantics) 오류 없이 올바른 기호 문자열과 관련된 의미

  • English

    • “Flying planes can be dangerous” 에는 많은 의미가 있을 수가 있음

  • Programming Language

    • 단 하나의 의미를 가지지만 프로그래머가 의도한 것이 아닐 수도 있음

문제가 발생하는 부분

  • Syntax 오류

    • 흔하며, 쉽게 발견하고 수정할 수 있음

  • Static semantics 오류

    • 일부 언어에서는 프로그램을 실행하기 전에 이를 확인함

    • 예측할 수 없는 동작이 발생할 수 있음

  • Semantics 오류는 없지만, 프로그래머가 의도한 것과 다른 의미

    • 프로그램이 충돌하고 실행이 중지됨

    • 프로그램이 영원히 실행됨

    • 프로그램이 답을 제공하지만 예상과 다름

Python Programs

  • program(프로그램)은 일련의 definition(정의)와 command(명령)로 구성됨

    • definition이 계산(evaluated)되고,

    • 쉘(shell)에서 python 인터프리터에 의해 명령(command)이 실행됨

  • commands(statements)은 인터프리터에게 어떤 일을 하도록 지시함

    • Shell에 직접 입력하거나 shell로 읽어 계산(evaluation)할 수 있는 파일에 저장할 수 있음

Objects

  • Program은 data object를 조작함

  • Object는 program이 object에 수행할 수 있는 작업의 종류를 정의하는 type이 있음

    • 안나는 인간이므로, 걷기, 영어 말하기 등이 가능합니다.

    • 츄바카(스타워즈 등장인물)는 우키(종족)라서 걸을 수 있어요. “으아아아아아“등의 말도 가능합니다.

  • Object는 크게 두 가지로 구분

    • Scalar(세분할 수 없음)

    • Non-scalar(액세스 할 수 있는 내부 구조가 있음)

Scalar Objects

  • int – integer(정수)를 표현 (예: 5)

  • float – float(실수)를 표현 (예: 3.27)

  • bool – True 또는 False 값을 가지는 ‘boolean’을 표시

  • NonType – 단일 None 값을 가지는 특별한 type

  • type() 으로 object의 type을 볼 수 있음

    >>> type(5)           <== Python shell에서 입력
int                   <== enter를 누르면
>>> type(3.0)
float

Type Conversions (cast)

  • Object를 다른 type으로 변환할 수 있음

  • integer '3’을 'float(3)'을 사용하여 float 3.0으로 변환

  • 'int(3.9)'는 float '3.9’를 integer '3’으로 표시

    <object> <operation> <object>

  • 우선 계산 식을 위해 괄호 ‘( )’를 사용

  • 괄호 없는 연산자 우선순위

변수에 값 binding(바인딩)

  • 동등 기호 ( = )를 사용하여 변수에 값 할당

    pi = 3.141592
pi_approx = 22 / 2

  • 값은 컴퓨터의 메모리에 저장됨

  • 할당(assignment)은 이름에 값을 바인딩 함

  • 변수의 이름을 사용하여 값을 돌려받음

Abstracting Expressions

  • Expression의 값에 이름을 지정하는 이유는?

  • 값 대신 이름을 재사용

  • 나중에 코드를 변경하기 쉬움

    pi = 3.14159
radius = 2.2
area = pi * (radius**2)

Programming vs. Math

  • 프로그래밍에서는, 𝒙 를 “풀이”하지 않음

    pi = 3.14159
radius = 2.2
# area of circle
area = pi * (radius**2)
radius = radius + 1

    • '=' : assignment(할당)

      • 오른쪽의 expression이 값을 계산

      • 변수 이름은 왼쪽

      • radius = radius + 1과 radius += 1은 같음

바인딩 변경

  • 새 할당문을 사용하여 변수 이름을 다시 바인딩 할 수 있음

  • 이전 값은 여전히 메모리에 저장되어 있지만 이에 대한 핸들은 손실됨

  • area 값은 컴퓨터에 다시 계산을 지시할 때까지 변경되지 않음

pi = 3.14159
radius = 2.2
area = pi * (radius**2)
radius = radius + 1
