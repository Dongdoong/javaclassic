tuples

  • 순서가 지정된 요소들의 순서로, 여러 타입의 요소가 있을 수 있음

  • 요소 값을 변경할 수 없음. immutable

  • 괄호로 표현

    te = () 	            # empty tuple
t = (2, "mit", 3)
t[0] 			        # 2
(2, "mit", 3) + (5, 6) 	# (2, "mit", 3, 5, 6)
t[1:2]                  # ("mit", )
t[1:3]                  # ("mit", 3)
len(t)                  # 3
t[1] = 4                # error! 개별 요소를 변경할 수 없음

활용

  • 변수 값을 swap(교환) 하는데 편리하게 사용

    x = y
y = x
    temp = x
x = y
y = temp
    (x, y) = (y, x)

  • 함수에서 하나 이상의 값을 return 하는데 사용

    def quotient_and_reminder(x, y):
    q = x // y # integer division
    r = x % y
    return (q, r)
(quot, rem) = quotient_and_reminder(4, 5)

채우기

  • tuple을 iterate(반복) 할 수 있음

    def get_data(aTuple):
    nums = () # empty tuple
    words = ()
    for t in aTuple:
        nums = nums + (t[0], ) # singleton tuple
        if t[1] not in words:
            words = words + (t[1],)
    min_n = min(nums)
    max_n = max(nums)
    unique_words = len(words)
    return (min_n, max_n, unique_words)

list

  • index를 통해 접근 가능한 순서화된 일련의 정보

  • list는 square brakets(대괄호)로 표시됨

  • list에는 element(요소)가 포함됨

    • 일반적으로 같은 유형의 요소

    • 여러 타입의 요소를 추가할 수 있음(일반적이지 않음)

  • list의 요소가 변경될 수 있으므로 list는 변경가능(mutable)

List operation - remove

  • del(L[index])를 사용하여 지정된 인덱스의 요소를 삭제

  • L.pop()를 사용해 list의 마지막 요소를 삭제, 삭제되는 요소를 반환

  • L.remove(element)를 사용해 지정된 요소를 삭제

    • 요소를 찾아 제거

    • 같은 요소가 여러 개일 경우 첫 번째 발견되는 요소를 제거

    • 요소가 목록에 없으면 오류 발생

      L = [2,1,3,6,3,7,0] # do below in order
L.remove(2)         # L이 변경됨 L = [1,3,6,3,7,0]
L.remove(3)         # L이 변경됨 L = [1,6,3,7,0]
del(L[1])           # L이 변경됨 L = [1,3,7,0]
L.pop()             # 0을 반환하고 L이 변경됨 L = [1,3,7]

List → string, string → list로 변환

  • list(s)를 사용하여 string을 list로 변환, string s 의 모든 character 요소로 된 list를 반환

  • s.split()을 사용하여 character parameter에 따라 string으로 분할할 수 있음.

    • parameter가 없으면 space(공백)으로 분할됨

  • ''.join(L)을 사용하여 character list를 string으로 변환

    • character를 추가하여 string으로 제공할 수 있음

      s = "I<3 cs"        # string s
list(s)             # ['I', '<', '3', ' ', 'c', 's'] 를 반환
s.split('<')        # ['I', '3 cs'] 반환
s.split('')         # ['I<3', 'cs'] 반환
L = ['a', 'b', 'c'] # list L
''.join(L)          # 'abc' 반환
"_".join(L)         # 'a_b_c' 반환

변경과 반복

Python tutor에서 수행할 것

  • List를 반복하면서 list를 변경하지 말 것

def remove_dups(L1, L2):
    for e in L1:
        if e in L2:
            L1.remove(e)

L1 = [1,2,3,4]
L2 = [1,2,5,6]

remove_dups(L1, L2)
def remove_dups(L1, L2):
    L1_copy = L1[:]
    for e in L1_copy:
        if e in L2:
            L1.remove(e)

L1 = [1,2,3,4]
L2 = [1,2,5,6]

remove_dups(L1, L2)

  • L1은 [3,4]가 아닌 [2,3,4]. 왜?

    • Python은 내부 카운터를 사용하여 루프에 있는 인덱스를 추적함

    • 변형하면 list 길이가 변경되지만 Python은 카운터를 업데이트하지 않음

    • 루프는 요소 2를 볼 수 없음
